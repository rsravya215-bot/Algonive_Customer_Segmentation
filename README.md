**Project Title:**

Customer Segmentation Using K-Means Clustering

** Project Overview:**

This project implements a Customer Segmentation System using K-Means clustering to categorize customers based on Age, Annual Income, and Spending Score.

The objective is to help businesses:

Identify premium customers

Target marketing strategies

Improve customer engagement

Increase revenue optimization

**🛠 Technologies Used**

Python

Pandas

Seaborn

Matplotlib

Scikit-learn

Google Colab

** Project Workflow:**

Data Cleaning & Preprocessing

Exploratory Data Analysis (EDA)

Feature Scaling

Elbow Method to determine optimal clusters

K-Means Clustering

Cluster Visualization

Customer Insights Interpretation

**Key Insights:**

High Income & High Spending customers identified as Premium Segment

Low Income & Low Spending customers categorized as Budget Segment

Distinct customer behavior patterns observed
